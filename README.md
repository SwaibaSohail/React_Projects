# CV Builder

A live-preview CV/résumé builder built with React and Vite.

## Features

- Live CV preview as you type
- Sections for General Info, Summary, Education, Experience, Projects, and Skills
- Add, edit, and remove multiple entries per section
- Download your CV as a PDF

## Tech Stack

React · Vite · CSS

## Getting Started

```bash
git clone https://github.com/SwaibaSohail/React_Projects.git
cd React_Projects/cv-builder
npm install
npm run dev
```

## Downloading Your CV

1. Fill in your details
2. Click **Download PDF**
3. In the print dialog, select **Save as PDF** and uncheck **Headers and footers**
